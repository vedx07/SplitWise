## Installation & Setup
### 1 Clone Repo
```
git clone https://github.com/vedx07/SplitWise.git
cd SplitWise
```
### 2 Backend Setup
```
cd backend
npm install
node server.js
```
### 3 Frontend Setup
```
cd client
npm install
npm run dev
```

### Create .env file (if not already) in backend
```
MONGO_URL = mongodb://localhost:27017/splitwise_dummy_db
JWT_SECRET = your_jwt_secret_key
PORT = 3000
```
